Flag geometry

by Simon Burton

Geometry! We all remember this from school:
  1. Any two points determine a unique line.
  2. Any two lines intersect in at most one point.
By line here we are thinking of a straight line. Obviously things are going to be more complicated if we allow the lines to wiggle around.

There's plenty of ways to monkey around with axioms for geometry. In particular, if it weren't for parallel lines we would have a theory that treated lines and points the same way:

  1. Any two points determine a unique line.
  2. Any two lines determine a unique point.
This is called projective geometry: we added some more points at infinity so that those parallel lines have somewhere to meet.

Now that we have some axioms, we can try drawing some models that satisfy those axioms. Because we want to keep things simple, we can try looking for models with only a finite number of lines and points.

This is called the Fano plane. It has seven points and seven lines. There's a weirdo line that looks like a circle, but we need this guy so that all the axioms are satisfied. And you should check this yourself: for any two points there is a unique line (going through them), and for any two lines there is a unique point (where they intersect).

This is a schematic diagram: even though I drew blue lines and lines have infinitely many points on them, each of these lines only has three points.

Just to emphasize the abstract nature of this construction, here is another picture of the Fano plane. Each point is represented as a green dot, and each line is represented by a red dot. These dots have connections that show how each point of the Fano plane lives on three lines, and vice-versa:

Canvas Fail

So each one of these black connecting lines represents a relation between a point and a line of the Fano plane. This relation is very simple, it just states that a point lies on a line. Or, a line contains a point. It gets a bit clunky how english has two ways of saying this, so we just say instead that a point and a line are incident.
